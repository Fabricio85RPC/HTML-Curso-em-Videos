Tamanho de fontes

Medidas absolutas

cm-mm-in(polegada)-px-pt-pc
EX: font-size: 2px;

Medidas relativas

em-ex-rem-vw-vh-%
EX: font-size: 2em;

Peso, Estilo e Shorthand font

Dica para não confundir mais

  • weight
  • width
  • height

Font-weight(peso da fonte)
EX: font-weight:lighter; (normal,bold,bolder)
Ex: font-weight:100;(100 a 900)
Ex:font-style:italic;
Ex: font-size:3em;
EX: font-family:'work sams' ,'sans-serif';
DICA:Aperte esse comando Ctrl+espaço dentro da font entre : ;

Shorthand font(resumindo as fontes)

Essa e a sequencia para fazer o resumo das fontes
font-style -> font-weight -> font-size -> font-family
EX: font: italic bolder 3em 'work sans', sans-serif;
