ID em CSS

Linkando id de tags de HTML ao CSS

Para estilizar a pagina de forma personalizada, utilizamos id em tags.
EX: h1 id="principal"

Dentro de css coloque # e depois o nome escolhido na id escolhida.

Diferença entre id e class

id: Pode usar varios ids no seu projeto mas cada um com o seu nome.
id e para um tag como se fosse um cpf(então sobressai ao class)

class:e utilizado em varias tags.
class e como se fosse uma familia, e facilita para estilizar seu projeto
Lembrando que dentro do seu style ou CSS primeiro digite . e depois o nome da class
