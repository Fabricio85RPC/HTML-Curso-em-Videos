Como tornar imagens responsivas com CSS

Para tornar uma imagem responsiva, você precisa dar um novo valor à sua propriedade de largura. Em seguida, a altura da imagem se ajustará automaticamente.
Uma coisa importante a se saber é que você deve usar sempre unidades relativas para a propriedade de largura, como porcentagem, ao invés de unidades absolutas como pixels.

Centralizar e Diminuir imagem

Centralizar

Utilizar display: block; para depois usar margin: auto;

Diminuir

utilizar o img{width:50%;}

Resposividade de Video

Para deixar um video mais responsivo:

  1. E preciso envelopar com uma div e colocar uma class nele
  2. Em CSS fazer uma caixa para o video com background-color, margin
  3. Em CSS fazer uma caixa para o filho (>) da div que no caso do youtube e iframe
  4. Depois de ter criado as duas caixas em CSS na caixa div.video vai ser incluido o position:relativo; e em div.video > iframeposition: absolute;
  5. Abaixo de position: absolute; você pode colocar top,left, width e height.(Geralmente aqui e feito em % e não em px
